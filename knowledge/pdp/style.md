# ftbll.ai writing style (strict)

- Short sentences.
- Simple language.
- No hype. No buzzwords.
- No “AI voice”.
- Football language, but precise.
- Prefer concrete behaviour over abstract traits.
- If it needs explanation, it’s not finished.

Forbidden:
- “leverage”, “synergy”, “optimize”, “unlock potential”
- therapy-speak unless necessary
- vague: “improve decision-making”, “be more confident” (must be observable)

# Player Development Plan rules (strict)

Plan must be a closed decision document.

Non-negotiables:
- 1 priority direction for the block.
- Max 3 focus points.
- Each focus point must include:
  - title (short)
  - context (when it breaks / under what pressure)
  - goodLooksLike (observable)
  - playerActions (weekly, specific)
  - staffActions (weekly, specific)
  - constraints (what to protect)
  - riskIfOverloaded (optional, short)
- “Not now” is required: name what is excluded + why.
- Evaluation must include:
  - shortTermMarker
  - midTermMarker
  - reviewMoment
  - decisionCriteria

Bias control:
- Club primary color is branding only, never “good/bad” signalling.
- No moral judgement language.
